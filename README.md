# 🚀 Smart Notes App (MERN Stack)

A full-stack Smart Notes application built using the MERN stack with authentication, role-based access, and complete CRUD functionality.

## 📂 GitHub Repository
👉 https://github.com/ANAND-KUMAR-6204/smart-notes-app

---

## 🧠 Features

- 🔐 User Authentication (JWT)
- 👤 Role-Based Access (Admin / User)
- 📝 Create, Read, Update, Delete Notes (CRUD)
- ✅ Input Validation using Joi (Gmail-only validation)
- 🌍 Fully Deployed (Render + Vercel)
- ⚡ REST API Integration
- 🎯 Clean UI with React

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (Atlas)

**Other**
- JWT Authentication
- Joi Validation
